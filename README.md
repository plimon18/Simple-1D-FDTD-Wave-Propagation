# Simple-1D-FDTD-Wave-Propagation
This is a personal project of trying to re-learn the Electromagnetism and Quantum Physics from university and reapply the techniques into a Python project.
A Finite Difference Time Domain (FDTD) algorithm using Maxwells electromagnetic equations will be used in order to simulate wave propagation in one dimension.
## Fidelity
If in case I finish the project in free-space, with the wave behaving appropriately, I will also include a dielectric barrier that creates reflected and 
transmitted waves, where the energies and momentums can be independentally calculated from each pulse created. I will try my best to modularize, and hopefully
move to two dimensions if I feel satisfied with the one-dimensional code.
## Best Practices
The goal of this project is to truly create a program that professionally adheres to Pythons best practices. This shall include classes, modules, github
version control, etc. This, in turn, will produce code that is functional, readable, maintainable, and robust.
